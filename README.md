MD5.js
=========

MD5.js is calc MD5 hash.


# API Document

https://github.com/uupaa/MD5.js/wiki/MD5

# Install, Setup modules

```sh
$ git clone git@github.com:uupaa/MD5.js.git
$ cd MD5.js
$ npm install
```

# Test

```sh
$ npm test
```

# Minify

```sh
$ npm start

  or

$ node node_modules/uupaa.minify.js --keep --output ./lib/MD5.min.js ./lib/MD5.js
```
